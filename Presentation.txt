Key Components 

- Arduino Mega board.			- Fullsize Bread Board. 

- Keypad. 						- RFID Reader. 

- 16 x 2 LCD monitor.				- Solenoid lock. 

-  Buzzer. 						- IR Sensor.                               

 

Functionality 

The user starts the device by pressing the start key, then he is expected to enter the device’s hard-coded password correctly to open the lock and press exit. The user has three attempts before the buzzer alarm goes off. When the alarm is on it cannot be turned off until an admin RFID card is used to ensure security and reliability. If the IR sensor detects trespassing on the door, it will also sound the alarm. 
And to save the device power consumption, the LCD is turned off if 30 seconds pass with no actions. An admin can also bypass the password using their RFID card.
